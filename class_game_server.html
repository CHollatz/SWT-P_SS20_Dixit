This class represents a game server. The server starts automatically and loads the player prefab into the online scene.

override void GameServer.Start ( )
inline

Start is called before the first frame update. The server loads the player info of the local player, if:

  • the player is the host -> start server as host
  • the player is the client -> join server as client NOTE: This is not important for development.
